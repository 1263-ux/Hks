## 生成后处理流程

```
AI导出PNG
  ↓
用 Aseprite 打开（或免费在线工具 lospec.com/pixel-editor）
  ↓
1. 导入调色板 assets/palette.gpl → Image → Mode → Indexed
2. 手工清理杂色像素（AI常有1-2px噪点）
3. Spritesheet 对齐检查（帧之间偏移不超过2px）
  ↓
导出 PNG-8 索引色
  ↓
放到 assets/ 目录
  ↓
Phaser 加载代码（见 assets/sprite-spec.md）
```

## 提示词技巧

| 关键词 | 效果 |
|--------|------|
| `硬边缘` `hard edge` | 像素块分明，不模糊 |
| `有限色板` `limited palette` | 颜色干净，复古感 |
| `纯色背景` `solid background` | 方便去背景 |
| `精细像素` | 中高密度像素风 |
| `俯视视角` `top-down` | 适合你的游戏视角 |
| `spritesheet` `4帧` | AI 尝试生成多帧排列 |
| `32位像素` | 不是8位那种极简风 |
