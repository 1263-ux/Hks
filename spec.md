# 《山西古建保卫战》Spec.md

## 1. 项目简介

**《山西古建保卫战》** 是一款以山西古建筑保护为主题的快节奏肉鸽割草小游戏。

玩家扮演“古建守护者”，在古建基地周围移动、闪避并自动释放技能，抵抗从四面八方涌来的灾害怪物。怪物会攻击古建基地的不同结构血条，玩家通过击败怪物获得经验并升级技能。部分技能在造成伤害时，可以回复古建对应结构血条，形成“打怪越猛，修得越快”的爽快反馈。

本项目目标是做一个**短时间内吸引眼球、容易上手、反馈直接、适合展示和传播**的轻量小游戏。

---

## 2. 项目目标

### 2.1 核心目标

- 让玩家在 10 秒内理解玩法。
- 让玩家在 1 分钟内感受到割草爽感。
- 让玩家在 3 分钟内体验到技能成长和古建血条修复反馈。
- 使用山西古建保护主题作为包装，形成差异化记忆点。
- 系统简单，不做复杂流派，不做复杂文保模拟。

### 2.2 玩家需要记住的内容

玩家玩完后应能记住：

1. 这是一个保护山西古建的割草游戏。
2. 古建有不同结构血条。
3. 不同怪物会攻击不同结构。
4. 技能不仅能打怪，还能修复对应结构。
5. 技能名来自古建修补和保护技术。

---

## 3. 游戏类型

- 类型：肉鸽割草 / 动作生存 / 基地防守
- 视角：2D 俯视角
- 操作：键盘 / 手柄 / 虚拟摇杆均可适配
- 单局时长：3-5 分钟 MVP，后续可扩展到 8-10 分钟
- 核心玩法：移动、闪避、自动攻击、升级选技能、保护古建

---

## 4. 核心玩法循环

```text
进入关卡
↓
古建基地出现在地图中心
↓
怪物从四周生成并向古建或玩家移动
↓
玩家移动并自动释放技能
↓
击败怪物获得经验
↓
经验满后升级，选择技能
↓
技能变强，怪物变多
↓
保护古建结构血条不归零
↓
坚持到倒计时结束，获得胜利
```

---

## 5. 胜负条件

### 5.1 胜利条件

MVP 版本采用倒计时胜利：

```text
坚持 5 分钟，古建未被摧毁，即为胜利。
```

后续可扩展 Boss 关。

### 5.2 失败条件

以下任一条件触发则失败：

```text
玩家生命值归零
或
任意一个关键古建结构血条归零
```

MVP 推荐使用：

```text
任意古建结构血条归零 = 游戏失败
```

这样规则最直观，压力最明确。

# 8. 古建基地系统

## 8.1 古建基地概述

每个关卡中央有一座古建基地。古建基地由多个结构血条组成。

MVP 版本包含 4 条结构血条：

1. 木质结构
2. 石质结构
3. 砖瓦结构
4. 彩绘壁画

怪物攻击古建时，会根据自身类型扣除对应结构血条。

技能命中怪物时，可根据技能类型回复对应结构血条。

---

## 8.2 古建结构血条

### 数据结构示例

```ts
type BuildingStructureType = 'wood' | 'stone' | 'tile' | 'painting';

interface BuildingStructureHp {
  type: BuildingStructureType;
  currentHp: number;
  maxHp: number;
}
```

### 显示名称

| type     | 显示名称 |
| -------- | -------- |
| wood     | 木质结构 |
| stone    | 石质结构 |
| tile     | 砖瓦结构 |
| painting | 彩绘壁画 |

## 8.5 古建受击规则

怪物进入古建攻击范围后，开始攻击古建。

```text
怪物攻击间隔到达
↓
根据怪物类型选择对应结构血条
↓
扣除对应血条 HP
↓
如果任意血条 HP <= 0
↓
游戏失败
```

---

## 8.6 古建回血规则

技能命中怪物时，如果该技能有修复效果，则回复对应结构血条。

```text
技能命中怪物
↓
造成伤害
↓
判断技能是否拥有修复效果
↓
回复对应古建结构血条
↓
血条不能超过 maxHp
```

# 9. 技能系统

## 9.1 技能系统概述

技能是玩家的主要攻击和修复手段。

技能分为：

1. 基础技能
2. 进阶技能

玩家必须先获得基础技能，之后才可以继续升级该技能。

每个技能最多 3 级。

```text
Lv.1：基础技能
Lv.2：进阶强化 1
Lv.3：进阶强化 2
```

---

## 9.2 技能获取规则

玩家升级时弹出 3 个技能选项。

选项生成规则：

1. 未获得的基础技能可以出现。
2. 已获得且未满级的技能可以出现升级选项。
3. Lv.2 需要先拥有 Lv.1。
4. Lv.3 需要先拥有 Lv.2。
5. 达到 Lv.3 后不再出现在升级池中。

---

```ts
type SkillId =
```

## 9.4 基础技能列表

MVP 只做 5 个技能：

1. 木构加固
2. 石材修补
3. 防水封护
4. 防虫处理
5. 彩绘修复

---

## 9.5 技能 1：木构加固

### 定位

前方直线冲击波，回复木质结构。

### 对应血条

```text
木质结构
```

### 技能等级

| 等级 | 名称     | 效果                                               |
| ---- | -------- | -------------------------------------------------- |
| Lv.1 | 木构加固 | 每 5 秒向前方发射一道木梁冲击波，造成 20 伤害      |
| Lv.2 | 梁柱补强 | 伤害提升至 30，命中怪物时回复 1 点木质结构         |
| Lv.3 | 榫卯强化 | 伤害提升至 45，冲击波宽度增加，命中回复提升至 2 点 |

### 实现方式

- 生成一个向玩家朝向移动的矩形/长条 projectile。
- projectile 可穿透敌人。
- 持续时间约 0.5-0.8 秒。
- 命中同一敌人一次即可。

---

## 9.6 技能 2：石材修补

### 定位

玩家周围范围震波，回复石质结构。

### 对应血条

```text
石质结构
```

### 技能等级

| 等级 | 名称     | 效果                                                   |
| ---- | -------- | ------------------------------------------------------ |
| Lv.1 | 石材修补 | 每 6 秒释放一圈石粉震波，范围 2.5，造成 18 伤害        |
| Lv.2 | 石粉填补 | 伤害提升至 28，范围提升至 3                            |
| Lv.3 | 表层加固 | 伤害提升至 40，命中怪物时回复 2 点石质结构，并轻微击退 |

### 实现方式

- 以玩家为圆心检测范围内怪物。
- 对范围内怪物造成伤害。
- 播放圆形扩散特效。
- Lv.3 额外击退怪物。

---

## 9.7 技能 3：防水封护

### 定位

圆形护罩扩散，克制酸雨怪，回复石质和砖瓦结构。

### 对应血条

```text
石质结构
砖瓦结构
```

### 技能等级

| 等级 | 名称       | 效果                                                  |
| ---- | ---------- | ----------------------------------------------------- |
| Lv.1 | 防水封护   | 每 7 秒释放水纹护罩，范围 3，造成 15 伤害             |
| Lv.2 | 排水导流   | 对酸雨怪伤害提升 100%，普通伤害 22                    |
| Lv.3 | 防渗保护层 | 命中酸雨怪时回复石质和砖瓦结构各 2 点，范围提升至 3.5 |

### 实现方式

- 以玩家为圆心释放圆形 AOE。
- 对酸雨怪使用额外伤害倍率。
- Lv.3 命中酸雨怪时回血。

---

## 9.8 技能 4：防虫处理

### 定位

持续药雾区域，克制白蚁怪，回复木质结构。

### 对应血条

```text
木质结构
```

### 技能等级

| 等级 | 名称     | 效果                                            |
| ---- | -------- | ----------------------------------------------- |
| Lv.1 | 防虫处理 | 每 6 秒释放药雾，持续 3 秒，每秒造成 8 伤害     |
| Lv.2 | 虫害清查 | 药雾持续 4 秒，对白蚁怪伤害提升 100%            |
| Lv.3 | 木构驱虫 | 药雾范围增加，命中白蚁怪时每秒回复 1 点木质结构 |

### 实现方式

- 在玩家当前位置生成一个圆形药雾区域。
- 区域持续数秒。
- 每秒对区域内怪物造成伤害。
- Lv.3 对白蚁怪触发回血。

---

## 9.9 技能 5：彩绘修复

### 定位

自动追踪最近敌人的颜料弹，回复彩绘壁画。

### 对应血条

```text
彩绘壁画
```

### 技能等级

| 等级 | 名称     | 效果                                                    |
| ---- | -------- | ------------------------------------------------------- |
| Lv.1 | 彩绘修复 | 每 4 秒向最近敌人发射颜料弹，造成 20 伤害               |
| Lv.2 | 颜料补绘 | 伤害提升至 30，命中后小范围爆炸                         |
| Lv.3 | 壁画护色 | 伤害提升至 40，命中回复 2 点彩绘壁画，颜料弹可弹射 1 次 |

### 实现方式

- 寻找距离玩家最近的怪物。
- 发射 projectile。
- Lv.2 命中后产生小范围 AOE。
- Lv.3 命中后回血并寻找附近第二目标弹射。

## 9.10 初始技能

MVP 推荐玩家开局自带：

```text
木构加固 Lv.1
```

原因：

- 开局立刻有攻击能力。
- 技能表现直观。
- 木构主题贴合第一关。

## 9.11 技能弹窗

玩家第一次获得技能或升级技能时，显示简短弹窗。

弹窗不做长篇科普，只显示：

```text
获得技能：木构加固

效果：
每 5 秒向前发射木梁冲击波。

小知识：
木构古建需要定期检查梁、柱、斗拱等构件。
```

升级时显示：

```text
技能升级：梁柱补强

效果：
冲击波伤害提升，命中时回复木质结构。
```

弹窗可点击关闭，也可以 2 秒后自动消失。

# 10. 怪物系统

## 10.1 怪物系统概述

怪物代表不同古建破坏因素。

MVP 版本设计 5 类怪物：

1. 白蚁怪
2. 风蚀怪
3. 酸雨怪
4. 火焰怪
5. 冻融怪

怪物数量不宜过多，重点是视觉表现清楚、攻击目标明确。

---

## 10.2 怪物基础数据结构

```ts
type MonsterType =
  | 'termite'
  | 'wind'
  | 'acid_rain'
  | 'fire'
  | 'freeze_thaw';

interface MonsterData {
  id: string;
  name: string;
  type: MonsterType;
  hp: number;
  speed: number;
  damage: number;
  attackInterval: number;
  expDrop: number;
  targetPriority: 'player' | 'building';
  attackStructures: BuildingStructureType[];
  controlEffect?: ControlEffectType;
}
```

---

## 10.3 控制效果

MVP 中控制效果保持简单。

```ts
type ControlEffectType =
  | 'slow'
  | 'knockback'
  | 'burn'
  | 'freeze';
```

| 控制效果  | 说明                         |
| --------- | ---------------------------- |
| slow      | 玩家减速                     |
| knockback | 玩家被轻微击退               |
| burn      | 玩家或古建持续掉血           |
| freeze    | 玩家短时间无法移动或大幅减速 |

# 11. 怪物 AI

## 11.1 基础 AI

```text

选择目标
↓
向目标移动
↓
如果碰到玩家，则对玩家造成伤害或控制
↓
如果进入古建攻击范围，则攻击古建对应血条
↓
死亡后掉落经验球
```

---

## 11.2 目标选择

MVP 大部分怪物优先攻击古建。

```text
targetPriority = building
```

如果玩家距离怪物很近，怪物可以临时碰撞伤害玩家，但不需要复杂追击逻辑。

---

## 11.3 古建攻击范围

古建基地周围设置一个攻击检测圈。

当怪物进入检测圈后：

```text
怪物停止移动
每隔 attackInterval 攻击一次古建
```

也可以让怪物继续贴近古建，进入更小范围才攻击。

# 12. 经验与升级系统

## 12.1 经验掉落

怪物死亡后掉落经验球。

经验球会停留在地面。

玩家进入拾取范围后自动吸附到玩家身上。

---

## 12.2 经验值

| 怪物   | 掉落经验 |
| ------ | -------- |
| 白蚁怪 | 1        |
| 风蚀怪 | 2        |
| 酸雨怪 | 3        |
| 火焰怪 | 4        |
| 冻融怪 | 5        |

---

## 12.3 升级所需经验

MVP 简单公式：

```text
ExpToNext = 10 + 当前等级 * 5
```

示例：

| 等级         | 升级所需经验 |
| ------------ | ------------ |
| Lv.1 -> Lv.2 | 15           |
| Lv.2 -> Lv.3 | 20           |
| Lv.3 -> Lv.4 | 25           |

---

## 12.4 升级流程

```text
玩家经验 >= 升级所需经验
↓
游戏暂停
↓
弹出 3 个技能选项
↓
玩家选择一个
↓
应用技能或升级
↓
关闭面板
↓
游戏继续
```


## 22.2 数值优先级

数值调试优先保证：

1. 前期玩家不会太快失败。
2. 中期怪物数量明显增加。
3. 后期画面足够热闹。
4. 技能升级后肉眼可见变强。
5. 古建回血效果要明显，但不能无限安全。

- 技能特效
- 怪物数量
- 古建血条变化
- 回血反馈
- 升级音效
- 击杀反馈
